# 12v-to-5v-Linear-regulator-
📖 About the PCB

After building the circuit on a breadboard, I designed a PCB to make it more stable and permanent. The PCB helps reduce loose connections and gives a neat, professional look to the circuit.

🧰 Components Placement (My Approach)

While designing the PCB, I placed components in a simple and logical way:
The DC input terminal is placed at one side of the board
The 7805 regulator is placed in the center
The output terminal is placed on the opposite side
Capacitors are placed close to the regulator pins for better filtering

🧭 PCB Design Steps (What I Did)

1. Drew the schematic in PCB design software (like KiCad/EasyEDA)
2. Assigned footprints for all components (through-hole)
3. Placed components neatly on the board
4. Routed tracks manually for better understanding
5. Checked for errors (ERC/DRC)
6. Generated Gerber files

🧾 Conclusion

Designing the PCB helped me understand real-world circuit implementation. It improved reliability and made the project look professional. This is a good beginner PCB project to learn layout and routing.
